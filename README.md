## Hi there 👋

I'm a mechanical engineer by training, a business builder by experience, and a curious person by nature.

I came to software through a long detour that included vibration research, embedded systems projects, commercial building automation, new product development, run a bakery in Montréal, home renovation, and somehow ended up here — after I have done an AI/ML program at McGill.

## What you'll find in this repo

Honest answer: **messy prototypes**.

This is an active workshop, not a polished portfolio. You'll find things that half-work, tutorials, experiments, and projects at various stages.

A few things I've been building:

- **Invoice OCR** — a mobile app that scans invoices and fills a database automatically. Built for a bakery. Probably useful for any small business drowning in paper.
- **Article generator** — an LLM pipeline that generates a receipe, deployed on my own Raspberry Pi server. [Have a look!](https://editor.notremaison.app)
- **IoT monitoring** — an ESP32 device pushing live data from sensors a dashboard; a recreation of what I did on Microchip PIC.
- **RAG experiments** — a LLM chatbot with retrieval-augmented generation for my bakery customers.
